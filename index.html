OverviewÔÉÅ

The WBZ451 Curiosity Board is a compact, feature-rich development platform designed by Microchip for rapid prototyping of wireless applications. It integrates the WBZ451 SoC, which supports both Zigbee and Bluetooth Low Energy (BLE) protocols, making it ideal for IoT and smart home projects.

Name: ev96b94a

Architecture: ARM

Hardware DetailsÔÉÅ

The board integrates multiple components to support wireless development and debugging.

Core ComponentsÔÉÅ

  • WBZ451PE RF Module: BLE + Zigbee

  • PIC32CX-BZ2 MCU: ARM Cortex-M4F

  • PKOB4 Debugger: USB-based programming

  • MCP73871: Li-Ion/LiPo charger with power path

  • SST26VF064B: 64-Mbit QSPI Flash

  • MCP9700A: Analog temperature sensor

ConnectivityÔÉÅ

  • USB 2.0 via USB2512B Hub Controller

  • 10-pin ARM SWD header

  • mikroBUS‚Ñ¢ socket for expansion

Highlight

The RGB LED is PWM-controlled and can be used for status indication or demo effects.
